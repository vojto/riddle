# View categories

## Client -> Server
Method: GET  
Requires: Auth  
URL: `/categories/`  
Fields: None  

## Server -> Client
Method: JSON  
Examples:  
* Typical example

  ```json
    [
      {
        "name": "Biology",
        "id": 1
      }
      ,
      {
        "name": "Mathematics",
        "id": 2
      }
    ]
  ```
